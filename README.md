# INSTRUCTIONS FOR SETUP

1. clone the repository

```sh
git clone https://github.com/anil-itwala/dn24.git
```

2. Install laravel

```sh
cd dn24
composer install
```

3. Install assets & compile them

```sh
npm install
npm run dev 
```
4. add env variable to .env which is shared by email

5. Run laravel server

```sh
php artisan serve
```
